# Setup Instructions for Purple Portfolio

## Step 1: Install Dependencies

Open a terminal (PowerShell or Command Prompt) in the project folder and run:

```bash
npm install framer-motion lucide-react clsx tailwind-merge
```

## Step 2: Run Development Server

```bash
npm run dev
```

The site will be available at `http://localhost:3000`

## Step 3: Build for Production

```bash
npm run build
```

The built files will be in the `dist/` folder.

## Admin Access

Navigate to `/admin/` to access the admin dashboard.

## Features Implemented

- Hero section with glowing ring animation
- About section with glassmorphism stats cards
- Services section with pricing cards
- Portfolio gallery with filterable grid
- Testimonials carousel
- Booking section with price calculator
- Contact form
- Admin dashboard with analytics
- API routes for portfolio, bookings, testimonials, and pricing
